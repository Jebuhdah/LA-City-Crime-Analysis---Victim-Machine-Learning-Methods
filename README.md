# LA-City-Crime-Analysis---Victim-Machine-Learning-Methods
End-to-end machine learning project that classifies Los Angeles crime incidents using real-world data, 
comparing multiple models to evaluate predictive performance and practical challenges in crime data analysis.

LA Crime Type Predictive Modeling
This project develops a machine learning pipeline to classify crime types using publicly available crime incident data from Los Angeles. The objective is to explore how incident characteristics such as location, time, and contextual features can be used to predict crime categories and to evaluate the strengths and limitations of different classification approaches when applied to real-world crime data.
The project follows an end-to-end data science workflow, including data cleaning, feature engineering, model training, and performance evaluation. Special attention was given to handling noisy and incomplete records, which are common in administrative datasets such as crime reports.

Multiple classification models were implemented and compared, including:
Logistic Regression
Random Forest
k-Nearest Neighbors
Naive Bayes
Neural Networks
Models were evaluated using standard classification metrics such as accuracy, F1 score, confusion matrices, and ROC/AUC. The results highlight trade-offs between model complexity, interpretability, and predictive performance, as well as the challenges associated with imbalanced and real-world datasets.

This project demonstrates:
Data cleaning and preprocessing of real-world datasets
Feature engineering for structured incident data
Supervised machine learning workflows
Model evaluation and comparison
Interpretation of classification results
The goal of this project is not only to build predictive models but also to better understand the practical challenges of applying machine learning methods to crime data and similar administrative datasets.
